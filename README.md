# asdf-lazygit

[lazygit](https://github.com/jesseduffield/lazygit) plugin for [asdf](https://github.com/asdf-vm/asdf) version manager.

## Install

```
asdf plugin-add lazygit https://github.com/aniaan/asdf-lazygit.git
```

## Use

Check out the [asdf](https://github.com/asdf-vm/asdf) readme for instructions on how to install & manage versions of lazygit.

## Example

```bash
# Show all installable versions
asdf list-all lazygit

# Install specific version
asdf install lazygit 0.44.1

# Set a version globally
asdf global lazygit 0.44.1

# Set a version locally
asdf local lazygit 0.44.1
```
